Description

An interactive Dashboard hosted on Github Pages that allows users to explore trends in crime in Seattle through 2017 to early 2023. Users are able to filter through specific types of crime such as crimes against society, property crimes, and personal crimes.

Select different years to see how the distributions change over different years, and there is also a button to toggle a median income map that can be used to accompany the heat map. There is a pie chart on the side bar that shows users the distributions of the three types of crime measured that were mentioned above.

Features

  • Created using HTML, CSS, and JavaScript
  • Basemap is from Mapbox using the Mapbox API for their light theme OpenStreetMap
  • Utilized Python and JavaScript to create data cleaning scripts and reformat csv files to geojson.
  • Created the pie chart using the Plotly library in JavaScript
  • Crime data is from the Seattle Open Data Portal crime records dataset.
  • The median income choropleth map is from Best Neighborhood
